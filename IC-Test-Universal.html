IC.Test.Universal

Description

Helpers to program the “Universal module”. This is essentially a small, type-safe DSL to produce the small stack-based programming language interpreted by the universal canister.

This DSL is expression-based, not stack based; seems to suite all our needs and is simpler to work with.

This language is not stable; therefore there is no separarte documentation of specification than this file and `universal-canistersrc`
