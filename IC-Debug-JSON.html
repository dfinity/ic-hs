IC.Debug.JSON

Contents

  • Orphan instances

Description

This module defines ToJSON instances of the IC state.

We put them into their own module, despite the usual advise against orphan instances, to emphasize that these are there just for debugging purposes using `ic-ref`.

(Why JSON? Because Browsers render them nicely in a interactive display where you can open and collapse subcomponents – much easier to get this feature this way, compared to writing custom HTML output.)
