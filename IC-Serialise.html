IC.Serialise

Contents

  • Orphan instances

Description

This module defines Serialise instances of the IC state.

We put them into their own module, despite the usual advise against orphan instances, to emphasize that these are not part of the Ref module with its “reference implementation” status.

Also, orphan instances are kinda ok in applications.
